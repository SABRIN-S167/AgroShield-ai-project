# 🌾 AgroShield AI and AgriSense

AgroShield AI and AgriSense is a smart agriculture project that helps farmers detect crop issues and monitor agricultural conditions using AI.

---

## 🚀 Features

- 🌱 Crop disease prediction
- 📊 Agriculture dashboard
- 🤖 AI-based analysis
- 📱 Simple and user-friendly interface

---

## 🛠️ Technologies Used

- HTML, CSS, JavaScript
- React / Vite
- Node.js (for backend if connected)

---

## 📦 Installation

1. Clone the repository:
   git clone https://github.com/SABRIN-S167/AgroShield-ai-project.git

2. Go to project folder:
   cd agroshield-ai

3. Install dependencies:
   npm install

4. Run the project:
   npm run dev

---

## 🌐 Usage

- Open the project in browser  
- Click on **Open Agriculture Dashboard**  
- Use features to analyze crops  
- https://agroshieldai.netlify.app

---

## 📁 Project Structure

- src/ → main source code  
- public/ → static files  
- components/ → UI components
